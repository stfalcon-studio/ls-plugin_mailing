Обновление
----------

Выполнить к БД запросы из update*.sql cоответственно версии:
- для апдейта с версии ранее 0.3 выполнить update.sql и update_from3.0.sql
- для апдейта с версии 0.3 выполнить update_from3.0.sql.
- для апдейта с версии 0.4 выполнить update_from_0.4.0_to_0.4.1.sql

Сгенерировать хеш отписки для уже существующих пользователей, исполнив в консоли скрипт
plugins/mailing/include/cron/generate-unsub-hash.php.
